Python List Sorter

Python Implementation

def sort_list_alphabetically(input_list):
    """Sorts a list alphabetically in ascending order"""
    return sorted(input_list, key=lambda x: str(x).lower())

# Example usage:
items = ["apple", "Banana", "cherry", "Date", "elderberry"]
sorted_items = sort_list_alphabetically(items)
print(sorted_items)

How It Works

This Python program sorts a list alphabetically using the built-in sorted() function with a custom key that converts each item to lowercase for case-insensitive sorting.

Key Features:

  • Case-insensitive sorting (treats "Banana" and "banana" the same)
  • Handles mixed data types by converting all items to strings
  • Preserves the original list while returning a new sorted list
  • Uses Python's efficient Timsort algorithm

Example Usage:

fruits = ["Apple", "banana", "Cherry", "date"]
sorted_fruits = sort_list_alphabetically(fruits)
print(sorted_fruits)
# Output: ['Apple', 'banana', 'Cherry', 'date']
